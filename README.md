# English Trainer - Тренажер "Составь пары"

Интерактивный тренажер для изучения английских слов через сопоставление пар "английское слово - русский перевод".

## Описание

Тренажер помогает в изучении английской лексики через игровой процесс. Пользователю предлагается составить пары, кликая на слова в двух колонках: английские слова слева и русские переводы справа.

## Функциональность

- **Две колонки** с английскими и русскими словами
- **Случайный порядок** слов в каждой колонке при каждой загрузке
- **Визуальная обратная связь**:
  - Выделение выбранных элементов
  - Подсветка правильных пар зеленым цветом
  - Подсветка ошибок красным цветом
- **Блокировка интерфейса** на время показа ошибки
- **Счетчик** правильно сопоставленных пар
- **Уведомление** об успешном завершении упражнения
- **Кнопка "Начать заново"** для перезапуска игры

## Как пользоваться

1. Откройте файл `index.html` в браузере
2. Кликните на слово в левой колонке (английском)
3. Кликните на соответствующий перевод в правой колонке (русском)
4. Если пара правильная - слова подсветятся зеленым и станут неактивными
5. Если пара неправильная - слова подсветятся красным и можно попробовать снова
6. Продолжайте, пока все пары не будут сопоставлены
7. Для начала новой игры нажмите кнопку "Начать заново"
