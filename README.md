# english-trainer


# Техническое задание

## Тренажёр «Составь пары»

---

## 1. Общий интерфейс

1. Страница содержит заголовок с инструкцией пользователю:
   **«Составь пары»**.

2. Ниже располагается игровая область, состоящая из **двух вертикальных столбцов**, расположенных рядом:

   * левый столбец — слова на английском языке;
   * правый столбец — переводы на русском языке.

3. В каждом столбце отображается **одинаковое количество элементов** (по 5 элементов).

4. Порядок элементов в каждом столбце **перемешан случайным образом** при загрузке страницы.

---

## 2. Элементы списка

1. Каждый элемент представляет собой **кликабельную карточку** с текстом.
2. Все элементы имеют одинаковый размер и визуальный стиль.
3. Элемент может находиться в одном из следующих состояний:

   * обычное;
   * выбранное;
   * правильно сопоставленное;
   * временно подсвеченное как ошибка.

---

## 3. Поведение при выборе элементов

### 3.1 Первый клик

1. При клике на элемент он переходит в состояние **«выбран»**.
2. Выбранный элемент визуально выделяется.
3. В каждый момент времени может быть выбран **только один элемент**.

---

### 3.2 Повторный выбор в том же столбце

1. Если пользователь кликает на другой элемент **в том же столбце**, предыдущий выбор снимается.
2. Новый элемент становится выбранным.

---

### 3.3 Выбор элемента во втором столбце

1. Если после выбора элемента в одном столбце пользователь кликает на элемент в другом столбце, система выполняет **проверку соответствия пары**.

---

## 4. Проверка пары

### 4.1 Правильная пара

1. Если выбранные элементы соответствуют друг другу:

   * оба элемента переходят в состояние **«правильно сопоставлены»**;
   * элементы визуально выделяются как правильные;
   * элементы становятся **некликабельными**;
   * выбранное состояние снимается.

---

### 4.2 Неправильная пара

1. Если выбранные элементы не соответствуют друг другу:

   * оба элемента **временно подсвечиваются** как ошибка;
   * на время отображения ошибки **все клики блокируются**;
   * после короткой задержки:

     * подсветка ошибки убирается;
     * элементы возвращаются в обычное состояние;
     * элементы остаются доступными для повторного выбора;
     * выбранное состояние снимается.

---

## 5. Ограничения и правила взаимодействия

1. Пользователь **не может взаимодействовать** с элементами, которые уже правильно сопоставлены.
2. Во время отображения ошибочной подсветки **никакие действия пользователя не обрабатываются**.
3. Пользователь может совершать **неограниченное количество попыток** до правильного сопоставления всех пар.

---

## 6. Завершение упражнения

1. Упражнение считается завершённым, когда **все пары сопоставлены правильно**.
2. После завершения все элементы находятся в состоянии «правильно сопоставлены».
